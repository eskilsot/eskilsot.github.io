Custom Laptime Simulator

Motivation

In order to evaluate the ideal motor choice and pack capacity, and to quantify the importance of vehicle parameters such as aerodynamics, a laptime simulator was used. I made a custom laptime simulator using python (and version controlled through github) to develop the simulations. This custom simulator allowed for complete control over the simulation, and allowed for easy automation of simulations, compared to commercial laptime simulators.

Features

The program was built using python, as it was free for the team to use, and had features that made the program easier to write and understand. The programs supports multiple vehicles and tracks. The vehicle parameters are imported in csv files which are easily editable, and the format for track data allows accelerometer data to be used (with the help of an online converter). The program can perform a sweep of any vehicle parameters, across multiple vehicles, and any track. Vehicle variables (such as velocity and acceleration) can be viewed over the duration of a lap. The laptime data can also be converted to a competition score
